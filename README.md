# 📊 Stock Market Financial Data Analysis

## 📌 Project Overview
This project analyzes historical stock market data for Apple (AAPL), Microsoft (MSFT), and Tesla (TSLA) using Python.

It focuses on understanding stock performance, risk (volatility), and daily returns using real financial data from Yahoo Finance.

---

## ⚙️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- yfinance
- Excel

---

## 📊 Key Features
- Collected real stock data using Yahoo Finance
- Cleaned and processed financial datasets
- Calculated daily returns using percentage change
- Measured risk using volatility (standard deviation)
- Visualized stock price trends and returns
- Exported results to Excel for reporting

---

## 📈 Key Insights
- Microsoft showed the highest average return in the dataset
- Tesla had the highest volatility (highest risk)
- Apple showed stable and consistent performance

---

## 📂 Output
- Clean dataset in Excel
- Visual charts for stock trends
- Risk and return analysis

---

## 🚀 How to Run
1. Install required libraries:
   pip install yfinance pandas numpy matplotlib openpyxl

2. Run the Jupyter Notebook:
   stock_analysis.ipynb
